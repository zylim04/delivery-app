SHAP Key Findings

  • vehicle_condition — vehicle condition affects rider reliability.
About SHAP

SHAP (SHapley Additive exPlanations) is a game-theory based method for explaining ML predictions.

Unlike basic feature importance, SHAP accounts for feature interactions and gives mathematically rigorous explanations.

Values are mean absolute SHAP values — direction of impact is ignored, only magnitude is shown.
