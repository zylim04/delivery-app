About This Project

📌 Project Overview

This system predicts food delivery time using machine learning. It was developed as a Final Year Project (FYP) using a dataset of 42,592 real delivery records from the Zomato platform.

The system uses XGBoost — a gradient boosting ensemble model — tuned with RandomizedSearchCV and 5-fold cross-validation to achieve a Test R² of 0.8393 and Test RMSE of 3.78 minutes.

🎯 Research Objectives
  1. Identify key factors that influence food delivery time using feature importance analysis.
  2. Build a leakage-free preprocessing pipeline for reliable model training.
  3. Compare multiple ML models and select the best performing model.
  4. Deploy the best model as a web-based prediction system.

📂 Dataset

  • After cleaning: 42,592
  • Features: 20 input variables
  • Target: time_taken_min
  • Train split: 80% (34,073 rows)
  • Test split: 20% (8,519 rows)
🏆 Best Model
  • Model: XGBoost
  • Tuning: RandomizedSearchCV
  • CV: 5-fold
  • Test RMSE: 3.7771 min
  • Test R²: 0.8393
